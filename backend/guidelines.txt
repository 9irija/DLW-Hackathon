Hybrid Merge-Insertion Sort — Features & Analysis
Overview
This Python program implements and analyzes a hybrid sorting algorithm combining merge sort and insertion sort.
* Merge Sort handles large arrays efficiently (O(n log n) complexity).
* Insertion Sort handles small subarrays efficiently (O(n^2) worst-case, but fast on small sizes).
* Hybrid Approach: Subarrays smaller than a threshold S use insertion sort, while larger subarrays are recursively processed with merge sort.
The code also performs comparative analysis of key comparisons and execution times for both the hybrid and pure merge sort algorithms.
________________


Implemented Sorting Functions
1. Merge Sort
mergeSort(arr, l, r)


* Recursive divide-and-conquer sorting algorithm.
* Splits array into halves, sorts recursively, then merges.
* Returns the number of key comparisons during sorting.
2. Insertion Sort
insertionSort(arr, l, r)


* Sorts a subarray from l to r.
* Counts key comparisons for analysis.
* Efficient for small arrays or nearly sorted data.
3. Hybrid Merge-Insertion Sort
mergeInsertionSort(arr, l, r, S)


* Combines merge and insertion sort.
* If subarray size ≤ S, uses insertion sort; otherwise uses merge sort recursively.
* Returns total key comparisons.
